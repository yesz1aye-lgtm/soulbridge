## 1. 품의결의서 템플릿 플레이스홀더 삽입 (우선순위: 높음)

**상황**  
`templates/품의결의서/raw.hwp` — 366호 실제 예시 파일이 들어있음.  
`templates/품의결의서/template.hwp` — 현재는 raw 복사본이며 아직 `{{플레이스홀더}}`가 삽입되지 않은 상태.

**필요 작업**  
`scripts/make-template-품의결의서.mjs` 스크립트를 실행해 template.hwp에 아래 플레이스홀더를 실제로 삽입할 것.

| 플레이스홀더 | 설명 | 위치 |
|---|---|---|
| `{{증제번호}}` | 예: `2025-366` | 품의서·결의서 헤더 |
| `{{시행일자}}` | 예: `2025. 12. 10` | 품의서 하단 |
| `{{품의_제목}}` | 품의서 제목 한 줄 | 품의서 상단 |
| `{{품의_예산분류}}` | 예산 과목 분류 줄 | 품의서 본문 |
| `{{품의_본문}}` | 품의서 주요 내용 | 품의서 본문 |
| `{{지출과목_카테고리}}` | 예: `교과목강사/강사수당` | 결의서 지출건명 셀 |
| `{{지출건명}}` | 결의서 지출 건명 | 결의서 지출건명 셀 |
| `{{관}}` | 예: `서울시교육청지원금` | 결의서 지출과목 |
| `{{항}}` | 예: `교육운영비` | 결의서 지출과목 |
| `{{목}}` | 예: `강사수당` | 결의서 지출과목 |
| `{{세목}}` | 있을 때만 (없으면 빈 값) | 결의서 지출과목 |
| `{{지출일}}` | 예: `2025.12.15` | 결의서 지출일·이체실행일 |
| `{{지출방법}}` | 예: `□카드결제  계좌이체` | 결의서 지출방법 |
| `{{금액_한글}}` | 예: `일금 칠백사십이만...원정` | 결의서 금액 |
| `{{금액_숫자}}` | 예: `7,426,560` | 결의서 금액 |
| `{{내용}}` | 결의서 세부 내용 | 결의서 내용 셀 |
| `{{첨부서류}}` | 첨부 서류 목록 | 결의서 첨부서류 셀 |

**참고**
- 품의서 본문(`{{품의_본문}}`, `{{내용}}`)은 멀티 단락 셀 — replaceAll 적용 범위 확인 필요
- 지출일과 이체실행일이 동일한 경우가 많으나, 다를 경우 `{{이체실행일}}` 별도 플레이스홀더 고려

## 2. 행사계획서 템플릿 생성 (우선순위: 중간)

**상황**  
`templates/행사계획서/` 폴더만 있고 template.hwp 없음.  
`templates/2025-품의,결의서 392-404호 예시.hwp` 안에 행사계획서 양식이 포함되어 있음 (para3, para26).

**필요 작업**  
예시 파일에서 행사계획서 부분을 추출 → 플레이스홀더 삽입 → `templates/행사계획서/template.hwp` 저장.

| 플레이스홀더 | 설명 |
|---|---|
| `{{행사명}}` | 행사 이름 |
| `{{일시}}` | 날짜 |
| `{{장소}}` | 장소 |
| `{{참여인원}}` | 학생 수 및 교사 |
| `{{목적}}` | 행사 목적 |
| `{{방침}}` | 지도 방침 |
| `{{실시_세부계획}}` | 세부 계획 |
| `{{기대효과}}` | 기대 효과 (있을 경우) |
| `{{예산}}` | 예산 |

## 3. fill-hwp.mjs 멀티 단락 셀 처리 개선 (우선순위: 낮음)

**상황**  
현재 `fill-hwp.mjs`는 replaceAll 기반으로 단순 문자열 치환만 함.  
품의서 본문·결의서 내용처럼 여러 단락으로 구성된 셀은 replaceAll로 전체 교체가 안 될 수 있음.

**필요 작업**  
`deleteRangeInCell` + `insertTextInCell` 조합으로 셀 전체를 새 텍스트로 교체하는 헬퍼 함수 추가 검토.
